📝 Comment ajouter un réceptif ?

  1. Créer un compte dans Firebase Console → Authentication (email + mot de passe)
  2. Copier l'UID du compte créé
  3. Cliquer + Ajouter un utilisateur ci-dessus et renseigner l'UID, le rôle, les fiches et les permissions
  4. Partager l'email + mot de passe au réceptif
